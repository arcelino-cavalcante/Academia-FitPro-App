Com certeza! É uma excelente ideia, e na verdade é muito mais simples (e de graça) do que construir um servidor complexo.

O que o João e outros projetos Open Source fazem se chama "Static API" (API Estática). O GitHub não roda bancos de dados pesados, ele simplesmente hospeda arquivos de texto (o seu exercicios.json) e as suas pastas de imagens/GIFs. Como o nosso App é super inteligente, ele só precisa do link desse JSON para baixar tudo!

Aqui está como funciona e como podemos criar a sua própria "API" mundial de exercícios hoje mesmo:

Como funciona uma "API" no GitHub:
O Repositório: Você cria um repositório Público (Ex: seu-nome/api-exercicios-brasil).
A Estrutura de Pastas: Dentro dele, você faz upload da sua pasta de GIFs (ex: /gifs/supino.gif) e um arquivo mestre database.json.
O Arquivo JSON: Dentro desse JSON, você lista os exercícios apontando para o link dos GIFs no próprio GitHub:
json
[
  {
    "id": "supino-reto",
    "nome": "Supino Reto com Barra",
    "musculo": "Peito",
    "animacao": "https://raw.githubusercontent.com/seu-nome/api-exercicios-brasil/main/gifs/supino.gif"
  }
]
O Segredo (Raw Links ou GitHub Pages): O GitHub te dá um link direto chamado raw.githubusercontent.com. Quando o seu App da Academia (ou qualquer app de outro desenvolvedor que quiser usar sua API) bater nesse link, ele recebe os dados na hora em alta velocidade, usando os servidores globais e gratuitos da Microsoft/GitHub.
Como podemos fazer isso agora:
Como você já tem uma pasta cheia de GIFs prontos, o que nós podemos fazer é:

Criar um Script Organizador: Eu posso escrever um script rápido no seu computador que lê automaticamente a sua pasta de GIFs, extrai os nomes dos arquivos, e gera o database.json perfeitamente estruturado para você em segundos (sem você ter que digitar um por um).
Subir para o GitHub: Depois, você só precisa arrastar essa pasta gerada para um novo repositório no seu GitHub.
Conectar ao App: Por fim, eu altero o nosso App para parar de usar banco local e passar a "puxar" os treinos direto da sua nova API na nuvem!
